        Available Endpoints

        POST /api/users/register

        Register a new user with first name, last name, email, and password.

        POST /api/users/login

        Login with email and password to receive an authentication token.

        GET /api/users/profile

        Get the current user's profile information. Requires authentication.

        PUT /api/users/profile

        Update the current user's profile information. Requires authentication.
